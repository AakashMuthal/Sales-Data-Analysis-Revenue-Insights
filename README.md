# 📊 Sales Data Analysis & Revenue Insights

## 📌 Project Overview
This project analyzes retail sales data using Python to generate business KPIs, revenue trends, and actionable insights.

The goal is to help businesses understand revenue patterns and improve decision-making.

---

## 🚀 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset
Retail sales dataset containing:
- Date
- Region
- Product
- Category
- Units Sold
- Unit Price

---

## 📊 Key Analysis
✔ Revenue Calculation  
✔ KPI Metrics Generation  
✔ Regional Sales Analysis  
✔ Product Performance Evaluation  
✔ Trend Analysis  
✔ Data Visualization

---

## 📈 KPIs Generated
- Total Revenue
- Total Units Sold
- Average Order Value

---

## 📊 Visualizations
- Revenue by Region
- Product Performance Chart
- Revenue Trend Line
- Category Heatmap

---

## 💡 Business Insights
- Electronics category generates highest revenue.
- North region shows strong purchasing behavior.
- Accessories sell high volume but lower revenue.
- Laptops contribute major profit margin.

---

## 📌 Business Recommendations
- Increase marketing in high-performing regions.
- Bundle accessories with electronics products.
- Focus promotions on high-revenue products.
